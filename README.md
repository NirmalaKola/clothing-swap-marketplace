# Clothing Swap Marketplace 👕

A full-stack MERN application where users can exchange clothes instead of buying new ones, promoting sustainable fashion.

## 🚀 Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## 📂 Project Structure

```
clothing-swap-marketplace
│
├── frontend
└── backend
```
